here is the old website pre redesign, bitter + sweet coffeeshop

i first identified usability problems by navigating through the website myself, and then I used WAVE and a Screen Reader to pick up on more issues presented via accessibility tests.

To address these usability concerns I decided to prioritize accessibility and visibility, while making sure that all my elements remained responsive. I wanted to be concise and true to the shop's own brand, but implement a clean design without any redundancies.
